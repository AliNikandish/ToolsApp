## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/ToolsApp.git
```

#### Go to the project directory

```bash
  cd ToolsApp
```

#### Install ToolsApp with npm

```bash
  npm install
```
#### Run:
```bash
  npm run dev
```
